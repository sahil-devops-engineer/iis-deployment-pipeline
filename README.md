# 🚀 CI/CD Pipeline: GitHub Actions to IIS Deployment

## 📌 Project Overview
This project demonstrates an automated CI/CD pipeline that deploys a web application from GitHub to a Windows IIS server using GitHub Actions and Web Deploy (MSDeploy).

The pipeline eliminates manual deployment steps and ensures faster, consistent, and reliable releases.

---

## 🧰 Tech Stack
- GitHub Actions
- Windows Server (IIS)
- Web Deploy (MSDeploy)
- YAML Pipelines

---

## ⚙️ Workflow

1. Code is pushed to GitHub repository  
2. GitHub Actions workflow is triggered  
3. Build process starts  
4. Deployment package is created  
5. Application is deployed to IIS server using MSDeploy  
6. Website is updated automatically
